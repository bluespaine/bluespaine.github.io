Play Music & Browse Music Info


유튜브 음악 재생과 음악정보 조회를 동시에 할 수 있는 서비스를 제공합니다.
아티스트와 트랙/앨범 제목을 통해 원하는 음악을 찾아 들을 수 있습니다.
음악 감상과 동시에 해당 트랙 혹은 앨범에 대한 자세한 정보도 얻을 수 있습니다.

1. Search Track



  • Search Track : 음악트랙의 아티스트와 트랙명을 검색창에 입력합니다.
  • 기본적으로 '영미권의 음악'을, '영어'로 검색하기를 권장합니다.
    (MusicBrainz는 해외의 음악 DB라, 한국음반 검색은 미흡)
  • 줄임말, 별칭이 아닌 공식적인 이름과 제목으로 검색하기 바랍니다.
  • 검색완료 시
    트랙 유튜브 영상(배급사 제공 공식음원 "topic", "provided to youtube" )
    해당 영상의 description
    MusicBrainz DB에서 제공된 트랙 세부정보(최초 및 공식 발매 기준)
    Lyrist.app에서 제공된 트랙 가사
    가 표시됩니다.
  • 결과가 부정확할 시 관사, 대소문자를 변경하거나 성-이름을 반대로 검색
  • 검색어가 부정확한 경우, 배급사 제공 음원영상이 없는 경우, 기타 DB에 트랙정보가 없는 경우 결과가 없거나 잘못 출력될 수 있음

2. Search Album



  • Search Album : 앨범의 아티스트와 앨범제목을 검색창에 입력합니다.
  • 기본적으로 '영미권의 음악'을, '영어'로 검색하기를 권장합니다.
    (MusicBrainz는 해외의 음악 DB라, 한국음반 검색은 미흡)
  • 줄임말, 별칭이 아닌 공식적인 이름과 제목으로 검색하기 바랍니다.
  • 검색완료 시
    앨범 유튜브 Playlist(배급사 제공 공식 음원영상 혹은 아티스트 공식채널의 앨범 재생목록이 아닐 수도 있습니다.)
    출력된 Playlist의 영상제목 리스트
    MusicBrainz DB에서 제공된 앨범 세부정보(최초 및 공식 발매반 기준)
    영문 WIKIPEDIA의 해당 앨범 문서
    가 표시됩니다.
  • 결과가 부정확할 시 관사, 대소문자를 변경하거나 성-이름을 반대로 검색
  • 검색어가 부정확한 경우, 해당 앨범단위의 Playlist가 없는 경우, 기타 DB에 앨범정보가 없는 경우 결과가 없거나 잘못 출력될 수 있음
